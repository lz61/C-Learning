8.数组和指针

事实上,数组名就是一种特殊的指针,但是依然有一定的区别,这个本节末尾讲.

9.数组在内存中存放是连续的.

代码示例:

```
int a[3]={1,2,3};
printf("a[0]=%x\n", &(a[0]));
printf("a[1]=%x\n", &(a[1]));
printf("a[2]=%x\n", &(a[2]));
```

再来Print一个:

```
printf("a=%x",a);
```

10.实例2:

```
int arr[5]={1,2,3,4,5};
int *p = arr; // arr就是一个指针,这里是让指针等于指针
printf("%d\n",*arr);
printf("%d\n",*p);
```

11.事实上,数组名是指向数组第一个元素的指针常量.

常量: 数组名的值不能被更改.

12.数组的索引,事实上就是一种指针的读取.

```
int arr[5]={1,2,3,4,5};
int *p = arr;
printf("数组第二个元素是%d\n",arr[1]); // 是2
printf("数组第二个元素也是%d\n",*(arr+1)); // 也是2
```

也就是说:

```
arr[n] == *(arr+n)
```

19.数组名与指针的区别:

```
int one=1,two=2;
int * a = &one;
a=&two;	// 可以更改指针类型变量的值
int a[1] = {1};
int b[2] = {1,2};
a=b;	// 不可以将b的值赋给a,因为a的值是一个常量.
```

数组名: 是一个常量,不能改变数组名.
