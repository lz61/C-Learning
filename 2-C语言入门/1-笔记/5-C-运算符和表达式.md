6.++、--运算符效果

```
int a=3;

int b = a++; // 先返回a,a再++;
// b=3,a=4;

int b = ++a; // 返回a+1
// b=4,a=4;

int b = a--;
// b=3,a=2;
int b = --a;
// b=2,a=2

3++ // 不行
```



7.逗号,的作用之一: 分割语句,顺次执行,返回最后一个表达式的结果.

```
int sum=5;
int pad=5;
pad = ++sum, pad++, ++pad; 
printf("sum=%d\n",sum);
printf("pad=%d\n",pad);
```

A.从左向右依次执行

B.整个表达式的结果是最后一个表达式的结果

答案:

```
// pad = ++ sum; pad++; ++pad;

// 在执行完第一句话之后的效果:
// pad=6,sum=6

// 三句话都执行完之后的效果:
// pad=8,sum=6
```
